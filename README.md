# HEMLOCK

A system independent package management core. (inspired by [SlackBuilds](https://slackbuilds.org))

## Build

Build requires:
- CMake >= 3.5
- libsqlite3-dev

~~~
mkdir build
cd build
cmake ..
cmake --build .
~~~
